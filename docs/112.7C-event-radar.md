# Event Radar
## Detecção Determinística de Eventos Cientificamente Significativos

Status: Draft
Versão: 1.0
Dependências:
- 112.7A Scientific Taxonomy
- 112.7B Research Profile
- ADR-007

---

# Objetivo

O Event Radar é responsável por responder exclusivamente à pergunta:

> "Este evento possui importância científica própria?"

Ele NÃO considera o perfil do pesquisador.

Ele NÃO calcula interesse.

Ele NÃO faz matching.

Ele NÃO prioriza pesquisas.

Sua única responsabilidade é medir a significância objetiva do evento.

---

# Motivação

Nem todo evento relevante para a ciência está presente no Research Profile.

Exemplos:

- erupção vulcânica inesperada
- grande terremoto
- nova missão espacial
- tempestade solar extrema
- guerra envolvendo infraestrutura científica
- rompimento de barragem
- novo satélite lançado
- colapso de geleira
- impacto de asteroide
- pandemia emergente

Mesmo que o pesquisador nunca tenha declarado interesse nesses assuntos,
o sistema deve ser capaz de perceber que eles merecem atenção.

# Separação de Responsabilidades

Event Radar

"Este evento é importante?"

Interest Engine

Pergunta:

As duas respostas são independentes.

Knowledge Event validado.

Não consulta banco.

Não consulta internet.

Não utiliza LLM.

Exemplos:

- erupção vulcânica
- poeira mineral
- tempestade severa
- ciclone extremo

## Astronomia

- asteroide
- explosão solar
- CME
- nova missão

## Oceanografia

- El Niño
- La Niña
- aquecimento oceânico
- bloom incomum

## Geologia

- terremotos
- falhas tectônicas
- subsidência

## Vulcanologia

- VEI elevado
- coluna eruptiva
- emissão SO₂

## Sismologia

- sequência sísmica
- megathrust
- tsunami

## Geopolítica

- fechamento de espaço aéreo
- conflito internacional
- sanções científicas

- lançamento de satélite
- perda de sensor

# Significance Score

Score contínuo.

```
0.0

↓

1.0
```

Não depende do pesquisador.

---

# Score Explicável

Toda decisão deve produzir justificativa.

Exemplo

```text
score = 0.92

Motivos:

• Grande erupção vulcânica

• Emissão elevada de SO₂

• Coluna acima de 20 km

• Possível impacto atmosférico global
```

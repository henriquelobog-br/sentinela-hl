# Decisões de Arquitetura (ADRs)

Registro formal e datado das decisões estruturais do Sentinela HL. Um ADR
aprovado é uma **restrição** do projeto — componentes futuros devem respeitá-lo.
Reabrir uma decisão exige um novo ADR que a supersede (status `Substituído`).

| [004](./ADR-004-contracts-first.md) | Contratos primeiro (models = SQL) | Aprovado |

Formato: Contexto · Decisão · Consequências.
